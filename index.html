Windows voice mouse app

声鼠

声で格子を表示し、番号・微調整・クリックでマウスを操作する Windows 用アプリです。

プライバシーポリシー サポート

できること

  • 「表示」で画面に格子を出し、番号でカーソルを移動できます。
  • 「右」「左」「上」「下」で細かく位置を調整できます。
  • 「クリック」「右クリック」「ダブルクリック」でマウス操作できます。
  • 「スクロール」「戻る」「リロード」など、よく使う操作も声で実行できます。

初回リリースについて

初回リリースは日本語音声認識を前提にした Windows 版として準備しています。音声認識には Windows の音声認識機能を使用します。
